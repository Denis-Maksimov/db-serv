О программе

Эта программа предназначена для мониторинга температур сушки
Принцип её работы заключается в следующем:
В результате технологического процесса сушки
программное обеспечение "Valmatics" производства
компании "Valutec" сохраняет значения параметров сушки
в базе данных SQL. Базы данных представляют из себя
файлы формата *.db размером около 30 Мбайт.
Расшифровать данные файлы возможно с помощью
СУБД (систем управления базами данных),
работающих по спецификации SQL.
В силу того, что периодическая передача баз данных
на удалённый компьютер для дальнейшего извлечения параметров
несёт большую нагрузку на сеть, было принято решение
написать программу (веб-сервер), которая будет локально
выполнять запросы к базам данным и выдавать в локальную
сеть в виде html-страниц параметры температур.
В ходе написания программы были использованы библиотеки:
* SQLite-amalgamation версии 3.7.13 (полностью бесплатная
в том числе и для коммерческого использования)
* GTK+ 3.0 - библиотека для работы с графическим интерфейсом
данная библиотека распространяется по лицензии GPL v.3.0

Программа была написана на языке C++ и скомпилирована в
компиляторе MinGW. Первоначально писалась на Linux платформе
на ОС Debian 10 с целью портирования на Windows, т.о.
данную программу при необходимости легко перенести на
Linux - платформу.
